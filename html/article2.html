AMD Radeon Adrenalin Edition 18.11.1 Drivers

Author: Marek Czerniejewski | 20/11/2018

AMD had released a driver update for Radeon graphics cards in order to improve and optimize performance in the newest games, along with some general updates that will be mentioned below. The three big titles that are optimized in this update are Battlefield V, Hitman 2 and Fallout 76. If you own a Radeon card and are planning on, or already are playing one of those games then make sure to get this driver update. Here’s a list of improvements this update brings:

  • 3% performance increase in Hitman 2 using the RX 580 card in 1080p and medium video settings and DX11
  • 8% performance increase in Battlefield V using the RX Vega 64 in 2080p and ultra settings and DX11
  • 9% performance increase in Battlefield V using the RX 580 in 1080p and ultra settings along with DX11

Bugs that have been fixed:

  • Games are now working properly and don’t shut off on certain video settings when Radeon Overlay is turned on
  • Assassin’s Creed: Origins no longer freezes on Windows 7 desktops
  • Strange Brigade no longer shuts off randomly when game is in DX12 mode

Overall this is a good update that fixes some annoying bugs, allowing for a trouble free experience when playing the latest triple A titles. Enjoy!
